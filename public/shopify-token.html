Extraer Access Token

Introduce las credenciales de tu app privada de Shopify para obtener el token de acceso necesario para conectar tu tienda.

Encuéntralos en Shopify Admin → Configuración → Apps y canales de ventas → Desarrollar apps.

¿Cómo funciona?
Al hacer clic en "Autorizar" serás redirigido a Shopify para aprobar los permisos. Una vez autorizado, volverás aquí con tu token listo para copiar.
