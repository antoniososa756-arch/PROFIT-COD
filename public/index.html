Conectar tienda Shopify

×

Para conectar tu tienda con PROFITCOD necesitas autorizar una app desde tu propio Shopify. El proceso es seguro y tarda menos de 2 minutos.

Paso 1 · Crea una app personalizada en Shopify
Paso 2 · Descarga y ejecuta el conector
Paso 3 · Autoriza la app cuando Shopify lo pida
PROFITCOD nunca solicita tu contraseña de Shopify.

Paso 1 · Nombre de la app

En tu panel de Shopify, crea una app personalizada con este nombre:

PFCOD

El nombre es solo interno para tu tienda. Puedes usar PFCOD o PROFITCOD Connector.

Paso 2 · Configuración de la app en Shopify

Ahora completa la configuración básica de la app en Shopify exactamente como se indica a continuación.

1. App URL
Puedes poner cualquier URL válida. Recomendado:
https://profitcod.app
2. Embedded app
✔ Actívalo (marcar “Embed app in Shopify admin”)

3. Webhooks API version
Selecciona la versión más reciente disponible (ejemplo: 2026-01)

4. Scopes (permisos)
Añade estos permisos:
read_orders
5. Redirect URLs
Déjalo vacío por ahora.

6. Guarda los cambios
Pulsa Save / Release para continuar.

Paso 3 · API Key y Secret

Último paso dentro de Shopify.

1. Ve a Settings → API credentials

2. Copia estos valores
  • API key
  • API secret key
3. Guárdalos
Los necesitarás en el siguiente paso para conectar tu tienda con PROFITCOD.

✔ PROFITCOD nunca pide tu contraseña de Shopify

Paso 4 · Conectar tienda con PROFITCOD

Introduce los datos de la app que acabas de crear en Shopify. PROFITCOD se encargará del resto.

✔ Tus datos se usan solo para autorizar la conexión con Shopify
